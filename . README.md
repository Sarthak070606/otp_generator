# NyayaBot OTP Email Verification

This is a simple Flask application to send OTP (One-Time Password) via Gmail and verify user email addresses.

## Features
- Enter your email to receive an OTP
- OTP is sent via Gmail (using App Password)
- Verify the OTP
- Simple Flask web interface

## Prerequisites
- Python 3.11+
- Gmail account
- Flask and Flask-Mail libraries
